# Java Modules Project

This project consists of two Java modules: `core-loop` and `llm`. Each module is designed to encapsulate specific functionality and can be developed and tested independently.

## Getting Started

To get started with this project, clone the repository and navigate to the project directory. You can build and run each module independently.

## Building the Project

Use your preferred build tool (e.g., Maven, Gradle) to build the modules. Ensure that you have Java 11 installed.

## Testing

Each module contains its own set of tests. You can run the tests using your build tool of choice.
